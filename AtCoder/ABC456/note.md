# 新しく学んだこと

## 文字列の部分列

- まず連続する文字を含まない場合を考え、そのときはすべての部分文字列が条件を満たす→作れる部分文字列の総数を考えればOK
  - 部分文字列は先頭の左端と末尾の右端を加えた $|S| + 1$ この中から2つの場所を持ってくることと同じ ( ${}_{|S|+1}\mathrm{C}_{2}$ 通り )
- 連続する文字列がある場合は、その部分で列を区切る ( $abbc$ だと $ab|bc$ のように)
- 分かれた文字列ごとに、連続する文字を含まない場合の計算方法で部分文字列の総数を出し、足せばOK
