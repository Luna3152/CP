# 新しく学んだこと

## 累積和

- $A[1] + A[2] + \cdots + A[N]$とあったときに、これらの部分和（$A[3]$ ~ $A[5]$の和など）を出したいときは、累積和を考える。
- $S[i] = A[1] + ... + A[i]$ としておくと、$A[3] + A[4] + A[5] = S[5] - S[2]$ですぐに求まる。

## i mod j があれば分解してみる

- $i = q * j + (i \mod j)$ より、 $i \mod j = i - q * j$
- 上を下に、シグマの中身を書き換えると、累積和が使える形になると気づく。
