# 新しく学んだこと

## ヒープ
- ヒープは最大値、最小値に対してなにか操作をしたい場合に効率的に実行できる。
- 今回の場合だと、C問題がヒープの練習問題。
- イメージとしては以下のような感じ
```
          1
         / \
        3   4
       / \  /
      7   6 5
```
- ヒープは上のような完全二分木
  - 上から順番に詰まる
  - 左から順番に詰まる
- pythonだと次のパッケージをimportするとheapを使える
```
from heapq import heappop, heappush
```
  - heappop が heap で出力（デフォルトだと最小ヒープだから0番目が pop）
  - heappush は heap になるように配列に push していく
- ヒープは完全二分木だから一次元配列で実装可能。また、配列のインデックスで親子関係が決まる。
  - 親: i
  - 左の子: 2*i + 1
  - 右の子: 2*i + 2
